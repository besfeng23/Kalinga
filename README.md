# Kalinga Dogs Philippines Foundation Website

Pre-launch website for **Kalinga Dogs PH**, a proposed Philippine animal welfare and rabies-prevention initiative focused on humane street-dog care, sterilization, adoption, LGU partnership, and transparent donor reporting.

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Deploy on Vercel

1. Import this repository in Vercel.
2. Framework preset: **Vite**.
3. Build command: `npm run build`.
4. Output directory: `dist`.

## Important Positioning Notes

This is a **pre-launch partnership website**. It intentionally avoids:

- donation checkout
- fake registration claims
- fake donor logos
- fake partner logos
- fake impact numbers
- public fundraising language before legal compliance

The current website is meant to recruit founding partners, veterinarians, LGUs, corporate CSR contacts, land/site partners, institutional funders, and professional volunteers.
